# WS6 Invariants (Draft)

- **MIG-01 Policy/authority preservation**  
  Migration MUST preserve policy and authority bindings and make them enforceable on the target.

- **MIG-02 Governance metadata preservation**  
  Data and evidence MUST move with governance metadata intact (jurisdiction, classification, retention, pointers).

- **MIG-03 Validation before execution**  
  Post‑migration validation MUST complete before governed execution resumes.

- **MIG-04 Evidence of migration actions**  
  Export, import, validation, cutover, and rollback/refusal MUST emit evidence.

- **MIG-05 Dependency disclosure**  
  Non‑substitutable dependencies affecting portability MUST be declared in a dependency manifest.
