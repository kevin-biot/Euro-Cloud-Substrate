# ECS Pilot Pack (Draft)

## Purpose
Provide a pilot-ready pack so providers and deployers can run a 30-day ECS validation with clear outputs.

## Pack contents
- Pilot definition template: `pilots/pilot-definition-template.md`
- Expected scorecard template: `pilots/scorecard-template.md`
- Day 0/7/30 execution plan: `pilots/day-0-7-30-plan.md`
- Example evidence bundles:
  - `docs/examples/evidence-bundles/baseline/`
  - `docs/examples/evidence-bundles/regulated-ml/`
  - `docs/examples/evidence-bundles/ncp/`

## Required pilot outputs
1. Declared profile claim and scope.
2. Export bundle with verifier inputs.
3. At least one refusal scenario with policy/authority anchors.
4. Completed scorecard with artifact paths.
