# Call for Participation

We are seeking contributors to define and harden the Euro Cloud Substrate (ECS).

## Who we need
- Senior European cloud architects
- Telco and sovereign cloud engineers
- Storage and Kubernetes experts
- Audit/evidence specialists
- Compliance-by-design practitioners

## How to engage
- Claim or co-lead a workstream (`ws*/`).
- Draft `spec.md`, `invariants.md`, `conformance-outline.md`, and diagrams.
- Record decisions in `decisions/`.
- Discuss in issues/PRs; keep artifacts in-repo.

## Initial workstreams
- WS1: Open Landing Zone (OLZ-EU)
- WS2: Euro Object Storage Contract (EOSC)
- WS3: Execution Envelopes
- WS4: Interop API Standards
- WS5: Evidence and Audit Plane
- WS6: Migration and Portability
