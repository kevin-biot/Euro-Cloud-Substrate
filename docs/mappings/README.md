# ECS Mappings Index (Draft)

## Purpose
Mappings are non‑normative crosswalks that relate ECS "how" (invariants, profiles, evidence) to public frameworks and programmes ("what"). They do not define certification or compliance labels.

## Foundational extraction
- **Cross-jurisdictional sovereignty minimum (normative extraction): `docs/mappings/cross-geo-sovereignty-minimum.md`**
  The minimum common substrate properties that every geo's sovereignty framework
  is implicitly requiring — EU, France, GCC, California, Colorado, China.
  This is the normative base. All other mappings sit above it.

## Reference implementation
- **SlapOS mapping: `docs/mappings/slapos-mapping.md`**
  SlapOS (Nexedi, France, 2010) as proof-of-existence that M1–M7 are implementable
  in production. Includes the Cloudwatt case study — the definitive example of
  institutional blindness selecting architectural incorrectness over architectural
  correctness in sovereign cloud procurement.
