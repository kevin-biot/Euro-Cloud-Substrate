# Conformance Checklists (Draft)

This folder contains draft, minimal conformance checklists for selected invariant families.
Conformance submissions should declare a selected evidence profile and include `evidence_profile_id`/`hash_profile_id` annotations in evidence bundles where applicable.

## Available checklists
- `dep.md` — dependency invariants
- `sup.md` — supply‑chain invariants
- `ops.md` — human/operations invariants
- `phy.md` — physical/connectivity invariants
